#### 安装 vite
```
npm install -g vite
```

#### 创建项目
```
npm init @vitejs/app
```
#### 运行
```
npm run dev
```

---

#### 安装 vue 的手脚架
```
npm install -g @vue/cli
```

#### 确保 vue cli 版本在 4.5.0 以上
```
vue --version
```

#### 创建项目
```
vue create test-vue
```

#### 运行
```
npm run serve
```

#### 打包生产
```
npm run build
```
