
#### 安装 python
> python 3.8.7

#### 安装 pipenv
> pip install pipenv

#### 创建虚拟环境
> pipenv install

#### 激活虚拟环境
> pipenv shell
>
> pipenv run

#### 虚拟环境
> pipenv --venv

#### 查看环境依赖包
> pipenv graph
>
> pip list

#### 安装 flask
> python -m pipenv install flask
> 
> pipenv install flask

#### 安装 管理环境变量
> python -m pipenv install python-dotenv
> 
> pipenv install python-dotenv
>
> 加载 .env 和 .flaskenv 中设置的环境变量
> 

#### 运行 flask
> python -m flask run
> 
> flask run
